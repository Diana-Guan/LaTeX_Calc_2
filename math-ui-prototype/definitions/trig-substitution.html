Trigonometric Substitution

Definition

Trigonometric substitution is a method for simplifying integrals that contain radicals by replacing \(x\) with a trig expression in \(\theta\).

Simple Example (Q34)

Since Q34 has \(\sqrt{4-x^2}=\sqrt{2^2-x^2}\), it matches \(\sqrt{a^2-x^2}\), so use \(x=2\sin\theta\).

Pattern Pages (click to open)

\(\sqrt{a^2-x^2}\Rightarrow x=a\sin\theta\)

\(\sqrt{a^2+x^2}\Rightarrow x=a\tan\theta\)

\(\sqrt{x^2-a^2}\Rightarrow x=a\sec\theta\)
