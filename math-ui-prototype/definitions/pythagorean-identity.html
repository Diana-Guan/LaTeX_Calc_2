Trigonometric Identities

Definition

Trigonometric identities are equations involving trigonometric functions that are true whenever both sides are defined. Common examples include identities relating sine, cosine, tangent, secant, cotangent, and cosecant.

Simple Example

For example, if an expression contains \(1+\tan^2\theta\), it can be rewritten as \(\sec^2\theta\). Likewise, \(1-\sin^2\theta\) can be rewritten as \(\cos^2\theta\).
