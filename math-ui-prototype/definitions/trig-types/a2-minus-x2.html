Pattern: \(\sqrt{a^2-x^2}\)

Definition

For \(\sqrt{a^2-x^2}\), use substitution \(x=a\sin\theta\).

Then \(\sqrt{a^2-x^2}=a\cos\theta\) (on the standard interval \(0\le\theta\le\frac{\pi}{2}\)).

Simple Example

\(\sqrt{9-x^2}\):

\(a=3,\quad x=3\sin\theta\)

\(\sqrt{9-x^2}=\sqrt{3^2-x^2}=3\cos\theta\)

Proof

If \(x=a\sin\theta\), then

\[ \sqrt{a^2-x^2} =\sqrt{a^2-a^2\sin^2\theta} =a\sqrt{1-\sin^2\theta} =a\sqrt{\cos^2\theta} =a|\cos\theta|. \]

On the usual trig-substitution interval \(0\le\theta\le\frac{\pi}{2}\), \(\cos\theta\ge0\), so this becomes \(a\cos\theta\).
