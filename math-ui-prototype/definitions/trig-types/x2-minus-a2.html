Pattern: \(\sqrt{x^2-a^2}\)

Definition

For \(\sqrt{x^2-a^2}\), use substitution \(x=a\sec\theta\).

Then \(\sqrt{x^2-a^2}=a\tan\theta\) (on the standard interval \(0\le\theta<\frac{\pi}{2}\)).

Simple Example

\(\sqrt{x^2-16}\):

\(a=4,\quad x=4\sec\theta\)

\(\sqrt{x^2-16}=\sqrt{x^2-4^2}=4\tan\theta\)

Proof

If \(x=a\sec\theta\), then

\[ \sqrt{x^2-a^2} =\sqrt{a^2\sec^2\theta-a^2} =a\sqrt{\sec^2\theta-1} =a\sqrt{\tan^2\theta} =a|\tan\theta|. \]

On the usual trig-substitution interval \(0\le\theta<\frac{\pi}{2}\), \(\tan\theta\ge0\), so this becomes \(a\tan\theta\).
