\(\sin^2\theta+\cos^2\theta=1\)

Definition

The fundamental trigonometric identity is

\(\sin^2\theta+\cos^2\theta=1\).

An equivalent rearranged form is \(1-\sin^2\theta=\cos^2\theta\).

Example (Q34)

Q34 uses this identity to rewrite \(1-\sin^2\theta\) as \(\cos^2\theta\).

Proof

On the unit circle, every point satisfies \(x^2+y^2=1\).

Using \(x=\cos\theta\) and \(y=\sin\theta\), substitute into that equation:

\[\cos^2\theta+\sin^2\theta=1\]

Reordering the terms gives:

\[\sin^2\theta+\cos^2\theta=1\]
