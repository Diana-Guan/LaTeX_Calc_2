Antiderivative

Definition

An antiderivative of a function is another function whose derivative equals the original function.

If \(F'(x)=f(x)\), then \(F(x)\) is an antiderivative of \(f(x)\).

Example

Since \(\frac{d}{dx}\left(\frac{x^3}{3}\right)=x^2\), one antiderivative of \(x^2\) is \(\frac{x^3}{3}\).
