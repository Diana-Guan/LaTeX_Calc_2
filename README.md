# LaTeX Calc 2

## Purpose
This repository contains my work in LaTeX. The main goal is to extend existing questions and solutions into more detailed, step-by-step explanations. The particular focus is on Question 34. The LaTeX structure, macro design, and visual style follow Alejandro’s work for Question 1.

## Project Structure
- `Q34.tex`, `Q34.pdf`: core Question 34 source file and rendered output
- `Calc_2/`: broader Calc 2 materials and a possible structure for expanding beyond Q34 into detailed solutions for all questions
- `Calc_2/Questions.tex`, `Calc_2/Questions.pdf`: question set and rendered output
- `Calc_2/Solutions/`: individual solution files
- `Calc_2/Progress.md`: project planning and progress notes

## How to Use
This repo is primarily a source and reference repository.

To work with the LaTeX files:
1. Open the `.tex` files in a LaTeX editor.
2. You can also work on them in VS Code by installing the LaTeX Workshop extension.
3. Compile the target file with your preferred LaTeX toolchain.
4. Use the included `.pdf` files to check the rendered outputs.
